### Python

Python : 동적 타이핑 언어

- 변수의 타입 명시 X
- 모든 걱이 객체이다
  - 클래스의 인스턴스로서 메서드를 가질 수 있음

```python
x = 10 # 변수의 타입 명시 X
print(x) # 10
# x는 int 클래스의 인스턴스이다
# 따라서, x는 int 클래스가 가진 메서드를 사용 할 수 있다.
print(type(x)) # <class 'int'>
```
