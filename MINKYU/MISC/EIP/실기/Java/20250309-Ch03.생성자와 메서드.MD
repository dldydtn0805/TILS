1. 기본생성자를 명시적으로 정의했을 경우

#### 1. 기본 생성자 명시

```java
class Person {
  // private : 작성된 곳 안에서만 알 수 있음(대충 일단 숨겨진 느낌)
  private String name;
  private int age;

  // 기본 생성자를 명시적으로 정의한 경우
  public Person() {
    this.name = "홍길동";
    this.age = 0;
  }

}
```
