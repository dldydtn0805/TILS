### Ch05. Java final 키워드

### final

final : 자바에서 상수 등의 변경 불가능한 값을 선언 할 때 사용하는 키워드

final을 사용하는 경우

1. 값에 사용

- 변수 값을 초기화 한 이후 변경 불가

2. 메서드에 사용

- 해당 메서드는 하위 클래스에서 오버라이드 불가

3. 클래스에 사용

- 해당 클래스는 상속 불가

4. 매개변수에 사용

- 해당 매개변수는 내부 블럭에서 값을 변경 불가

5. 객체 참조에 사용

- 해당 객체는 참조를 변경할 수 없음
- 내부 상태는 변경 가능

6. 초기화 지연

- final 필드는 한 번만 초기화 -> 생성자에서 값 초기화 가능

```java
class Parent {
  // 오버라이딩 불가능하게 설정(너 내꺼만 써 그냥 ㅋㅋ)
  public final void finalMethod() {
    System.out.println("하위 클래스에서 오버라이딩 불가능한 메서드");
  }
}

class Child extends Parent {
  // 이거 안됨
  // public void finalMethod() {
  //   System.out.println("하위 클래스에서 오버라이딩 불가능한 메서드");
  // }
}

// 상속 불가능하게 설정
final class finalClass {

}

// 이거 안됨
// class subClass extends finalClass {

// }

public class Main {
  public static void main(String[] args) {
    // 너는 이제부터 100이여~~(변경 불가능한 값 선언)
    final int MAX_VALUE = 100;


  }
}
```
