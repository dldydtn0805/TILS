# 클론 코딩

## Zoom

## Ch01. Zoom Clone Coding

### Zoom Clone Coding

Zoom : 화상 채팅 방에서 여러 사람이 모여 화상통화, 채팅 등의 기능을 사용할 수 있는 프로그램

Zoom을 클론 코딩 해 봅시다!

### 사전 준비

Zoom을 클론 코딩하기 위해 필요한 기능이 뭐가 있을까?

핵심 기능

1. 로그인(일반, 소셜)
2. 화상 채팅 방 들어가기
3. 화상 채팅 방 만들기
4. 화상 채팅 방 들어가서 인원들과 화상 통화 및 채팅

부가 기능

1. 프로필 꾸미기
2. 친구 추가
