아이스크림(영어: ice cream, 문화어: 얼음보숭이, 에스키모)은 냉동 디저트로, 보통 크림에 향신료와 거품을 낸 흰자위를 넣고 얼린 음식이다. 초코 맛, 딸기 맛, 바닐라 맛, 녹차 맛 등 여러가지의 맛이 있다. 참고로, ice cream은 iced cream 또는 cream ice에서 파생된 단어다. 하드 아이스크림와 소프트 아이스크림 등으로 나눌 수 있다.
