프론트엔드와 API

프론트엔드는 API를 통해 백엔드와 통신한다. 웹과 모바일 프론트엔드의 경우 API는 HTTP 요청/응답에 기반을 둔다. API는 백엔드 포 프론트엔드(Backend for Frontend, BFF) 패턴을 사용하여 설계되며 프론트엔트측에서의 처리를 용이하게 하기 위해 응답을 처리한다.
